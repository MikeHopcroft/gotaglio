# GoTaglip

## Building GoTaglio

1. Verify you have python version >=3.12. Note that 3.13 may not be supported yet.
1. `pip install poetry` outside of any virtual environment
2. `git clone gotaglio`
3. `cd gotaglio`
4. `python -m venv .venv`
5. `.venv\Scripts\activate`
6. `poetry install`
